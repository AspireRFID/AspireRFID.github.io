This section describes the tools to management the components of the Aspire middleware.

1 Context

Aspire' end-to-end management tools are availables as RCP standalone applications and as plugins for the VisualVM.
  • RCP based tools use WebServics for communication
  • VisualVM tools use the JMX specifications. JMX (Java Management eXtension) is the de-facto standard for managing and for monitoring Java applications and middlewares.
