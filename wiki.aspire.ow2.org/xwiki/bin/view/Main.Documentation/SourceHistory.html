Branches

AITdev

The AITdev branch is the initial effort by AIT to start development for the AspireRFID middleware. The main target of this branch is to provide software that follows the AspireRFID’s Middleware and Programmability specifications. Thus, this branch will be providing software components for:

  • Reader Core Proxy with RFID reader Connectors and simulators
  • Filtering and Collecting
  • Business Event Generator
  • Information sharing repository
  • Various connector implementations (ERP, WMS, RDBMS)
  • And the AspireRFID IDE for configuring, programming and managing the middleware.
This branch is maintained by the AIT team

The AITdev Branch have moved to the AspireRFID trunk and it will keep evolving from there.

RFID Suite

The RFID Suite precedes the Aspire RFID middleware. The source code is available on the forge (http://svn.forge.objectweb.org/cgi-bin/viewcvs.cgi/aspire/branches/rfidsuite).

The available documentation concerns :

  • the Specification datasheet (.pdf, .doc, wiki)
  • the Installation manual (.pdf, .doc, wiki)
  • the Administration manual (.pdf, .doc, wiki)
  • the User guide (.pdf, .doc, wiki)
