Sponsored Projects

Sponsored Projects are funded by AspireRfid Supported and Donators

Advanced Management Functionality for LLRP Readers

  • Project Title: Advanced Management Functionality for LLRP Readers
  • Contact Person: John Soldatos (jsol@ait.edu.gr), Nikos Kefalakis (nkef@ait.edu.gr)
  • Project Sponsor: TBA (To be announced)
  • Available Funding: TBA (To be announced)
  • Description: The proposed project will modify the current EPC-LLRP Filtering and Collection HAL to support the following management functionalities:
    • Give the ability to the user to configure reader's sensitivity and transmitted power per Antenna (% presentence of the received min and max values)
    • Give the ability to the user to configure the reader to start/stop reading and send report when a GPIO trigger is activated (again taking in consideration the readers capabilities)
    • Give the ability to the user to configure the reader to send reports at a stable time interval
    • Activate the data reported per request
      • Antenna source
      • peakRSSI
    • Set up a specific frequency to work.
Interested students/parties should contact the listed AspireRfid people for more details.
