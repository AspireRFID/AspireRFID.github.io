It implements several specifications from consortiums such as EPC Global, NFC Forum, JCP and OSGi Alliance. AspireRFID provides also a set of tools enabling RFID consultants to deploy RFID solutions without a need for tedious low-level programming. AspireRFID allows the specification of RFID enabled processes. Accordingly, the tools generate all the RFID artifacts required to deploy these solutions over the AspireRfid middleware.
